# Fileprocessing-hw

## How to upload work

1. Create venv
   ```powershell
   python -m venv .venv
   ```
   or
   ```powershell
   python3 -m venv .venv
   ```

2. Activate venv
+ For Windows:
  ```powershell
  .\.venv\Scripts\activate.ps1    
  ```
+ For Linux / macOS
  ```powershell
  source .venv\bin\activate
  ```

3. Clone Repositori
   ```bash
   git clone https://github.com/Yar00myr/Fileprocessing-hw
   ```
   or 
   ```bash
   git clone https://github.com/Yar00myr/Fileprocessing-hw.git
   ```
